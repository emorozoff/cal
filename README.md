# $horts

Интерактивный калькулятор дохода канала на **YouTube Shorts** от рекламы (AdSense).
Вписываешь число просмотров — получаешь оценку заработка диапазоном, с дублем в рублях
по актуальному курсу ЦБ РФ.

**Живая версия:** https://emorozoff.github.io/shorts-calc/

Один статический `index.html` без зависимостей и сборки — открывается по ссылке с телефона
и с ноутбука.

## Что умеет

- Оценка дохода в **день / месяц / год** в `$` и `₽`.
- Диапазон **пессимист / реалистично / оптимист** — RPM у Shorts сильно плавает.
- Пресеты RPM по нишам × множитель по географии аудитории (Tier 1/2/3).
- Продвинутый режим: свой RPM и курс вручную.
- «Доход на 1 млн просмотров» — человеческая единица для копеечного RPM.
- Обратный расчёт: сколько просмотров нужно ради $X в месяц.
- Курс USD→₽ подтягивается с ЦБ РФ (`cbr-xml-daily.ru`), с ручной правкой и запасным значением.
- Расчёт зашивается в URL — можно переслать ссылку с готовыми цифрами.

## Как считается

```
Доход в месяц ($) = Просмотров в месяц / 1000 × RPM_shorts
RPM_shorts = базовый_RPM_ниши × множитель_географии
```

RPM для Shorts крошечный (обычно $0.02–$0.10 за 1000 просмотров) — монетизация идёт из
общего пула Shorts, и YouTube забирает часть. Поэтому цифры — **оценка по порядку величины**,
а не гарантия дохода.

## Технически

Vanilla HTML/CSS/JS, ноль зависимостей. Чистые функции расчёта отделены от рендера —
пресеты ниш и множители легко поправить в начале `<script>`.
